# Hash Map Pattern

### Hash maps allow:
- O(1) average lookup time
- Tracking frequencies
- Index mapping

---

### Typical problems:
- First Unique Character
- Contains Duplicate
- Word Pattern
- Isomorphic Strings
- Two Sum

---

### Complexity:
- Time: O(n)
- Space: O(n)

---

Solutions in this folder demonstrate the use of dictionaries for efficient lookup and counting.
